## BurbFindr

Not to be confused with Grindr. BurbFindr is a tool to help you find a suburb in your city in your price range.

## Tech

Using:

* Ruby on Rails
* JS
* ajax
* jQuery
* Underscore
* Handlebars
* Google Maps

## Installation

* Clone repository
* Run `Rake DB:create`
* Run `Rake DB:migrate`
* Run `Rake DB:scrape`
* `bundle`
* Start on your local host with `rails s` and enjoy :trophy:
